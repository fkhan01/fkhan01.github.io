Computer science student at Queens College looking to design engaging user experience through software. I’ve developed a user interface using JavaFX for an application for detecting and deleting duplicate photos in a directory. In the process I’ve leveraged tools like Photoshop and Scene builder to produce modern-looking interfaces.

TIDY SNAPS

Detects and deletes duplicate photos in a directory (and sub-directories, if desired). Duplicates are detected using a perceptual hashing algorithm. Developed using Java 8 (streams, lambdas) and Java FX. Designed the interface of program using JavaFx,SceneBuilder and CSS.
